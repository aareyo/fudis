# How to start developing

<br />

## Set up the environment

Follow instructions in [GettingStarted](/docs/documentation-introduction-gettingstarted--page) to start developing Fudis

## Source code

Clone the repository [Fudis Design System](https://github.com/funidata/fudis.git) from GitHub.

```
git clone https://github.com/funidata/fudis.git

```

## Install dependencies - VAATII TARKASTUSTA

Install dependencies to ngx-fudis directory

> Move to directory and install

```
cd ngx-fudis && npm install

```

## Build application and watch for changes

All following command should be run from **the root** of the fudis repository

> Build and watch for changes in Fudis package

```
npm run dev

```

## Start sandbox application

> Open sandbox in browser http://localhost:4200

```
npm run sandbox

```

## Start Storybook

> Open storybook in browser http://localhost:6600

```
npm run storybook

```

## Run tests

This command will run unit tests

```
npm run test

```

## Formatting and linting

Fudis uses Husky Git hooks to prevent unformatted code being pushed to GitHub.
To execute linting check separately run

```
npm run lint
```

and

```
npm run style-lint
```

in the root of the repository.
