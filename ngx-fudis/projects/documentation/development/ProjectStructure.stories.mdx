## Foundation

Foundations directory contains all project foundation design tokens and mixins. Including tokens and mixins for e. g. borders, breakpoints, colors, spacing and typography.

## Themes TARVIII TARKENNUSTA!!

Themes folder includes theme definitions that are needed for Angular Material component styling. If Angular Material component requires overriding style customization, these styles are recorded in component specific scss file.
