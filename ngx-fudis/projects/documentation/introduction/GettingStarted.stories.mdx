# How to Start Using Fudis in Your Application

<br />

## Installation

You can find package information at NPM https://www.npmjs.com/package/@funidata/ngx-fudis

#### 1. Install Angular Material Peer Dependency:

```
ng add @angular/material

```

Choose these options when prompted:

<code>
	{' '}
	? Choose a prebuilt theme name, or "custom" for a custom theme: Custom <br /> ? Set up global Angular Material typography
	styles? No <br /> ? Set up browser animations for Angular Material? Yes
</code>

<br />

#### 2. Then Install Fudis:

```
ng add @funidata/ngx-fudis

```

#### 3. Import Core Styles

The core styles from Angular Material must be imported in your application for the components to work properly. The easiest way to do this is to have a styles.scss file in project root with the following content:

```
@use "@angular/material" as mat; @include mat.core();

```

> Fudis does not include the Material core styles for you because **they should be included exactly once** in your application.

#### 4. Check Fudis Import in `app.module.ts`

Check your application's `app.module.ts` file that Fudis is imported correctly.

```
import { NgxFudisModule } from 'ngx-fudis';

@NgModule({
	...
	imports: [
		...
		NgxFudisModule
	],
	...
})
```

#### 5. Define Application's Root Styles

##### Set Primary Colors
```
:root {
	--fudis-color-primary-light: #hex;
	--fudis-color-primary: #hex;
	--fudis-color-primary-dark: #hex;
	--fudis-color-primary-extra-dark: #hex;
}
```
##### Set Rem Base Value

By default Fudis rem values follow 8 pixel base values, when having default browser CSS font-size of 100%, 1 rem equals 2 * 8px = 16px.
To apply this to your application, add to the root CSS style file:

```
:root {
	--fudis-rem-multiplier: 1;
}
```
If application uses other rem base value by setting font-size to e. g. 62.5% . Adjust -fudis-rem-multiplier accordingly.

```
html {
 	font-size: 62.5%; // this sets 1rem to equal 10px
}

:root {
 	--fudis-rem-multiplier: 0.625;
}
```

#### 6. Include Fudis Foundation SCSS-field

In your project's root style file import Fudis foundation package. After that Fudis fonts should be loaded in Fudis components. If you want to use Foundation tokens and mixins in your own component styles, use them like in example below and check out documentation pages for Foundations.

```
@use '@funidata/ngx-fudis' as fudis;

.my-amazing-component{
	@include fudis.text-color('white');
	@include fudis.bg-color('primary');
	@include fudis.body-text-lg-regular;
	margin: fudis.spacing-md fudis.spacing-xs;

	&:focus{
		@include fudis.focus-generic;
	}

}

```

#### 7. Use Fudis Components in Your Project

```
import { ButtonComponent } from './button.component';

<fudis-button label="Eiku"></fudis-button>
```

or

```
import { Component } from '@angular/core';

@Component({
 selector: 'app-root',
 template: ` <fudis-button label="Eiku"></fudis-button> `,
 styleUrls: ['./app.scss'],
})
export class AppComponent {
 title = 'dev';
}
```
