# Introduction

Fudis Design System provides guidelines, design assets and components that work as the foundation for all of Funidata's products. The purpose of this Angular-based system is to support a sustainable and consistent development and accessible customer experience for our services.

## For product developers

<br />

### How to use Fudis in your application

To install Fudis in your application, check [How to start using Fudis documentation.](/docs/documentation-introduction-how-to-start-using-fudis--page).

For each component's documentation, usage guidelines and examples check **Docs** tab in component's Storybook page. E. g. [Docs tab for Button component](/docs/components-button--all-variants).

If you need support, do not hesitate to send message to Fudis developers and maintainers. Most convinient way for that is our [#fudis Slack channel](https://sisudev.slack.com/archives/C03G46K6LG1).
