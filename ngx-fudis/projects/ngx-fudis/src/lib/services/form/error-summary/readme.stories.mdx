# Error Summary

## Functions

### Reload errors

Reload errors is an error summary service functionality that updates the visible list of errors shown in the Fudis Error Summary component.

Import `FudisErrorSummaryService` and add the parameter to constructor:

```
import { FudisErrorSummaryService } from 'ngx-fudis';

...

constructor(
  private _service: FudisErrorSummaryService
) {}
```

When calling `reloadErrors` function, you can set an optional timeout delay in milliseconds:

```
this._service.reloadErrors(); // sets timeout to 0ms

this._service.reloadErrors(100); // sets timeout to 100ms
```
