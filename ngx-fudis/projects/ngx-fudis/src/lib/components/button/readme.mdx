# Button

Buttons are categorized based on their importance and functionality, such as primary, secondary, or tertiary buttons.
It's important to use the correct type of button for the intended action and to not overuse them to avoid overwhelming the user.

### Accessibility

- Button color contrast ratio meets AA and AAA levels
- Button has a visible focus state
- Disabled button can be focused. State is communicated for screen reader
- Button click can be triggered with keyboard <code>enter</code> and <code>space</code> key
