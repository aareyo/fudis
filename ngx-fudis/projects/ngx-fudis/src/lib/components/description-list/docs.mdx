# Description List Component

Description List displays a list of key value pairs and has two different list variants: `regular` and `compact`.
Description List extends Grid API and can take all the same inputs as Fudis Grid, see [Properties](#properties) table. Fudis Grid can be disabled in Description List by setting `disableGrid` as true.

## Usage Guidelines

Description List is built using child components as follows:

<Source
  code={`
<fudis-dl>
  <fudis-dl-item>
    <fudis-dt></fudis-dt>
    <fudis-dd></fudis-dd>
  </fudis-dl-item>
</fudis-dl>`}
/>

Term and Details content is passed through `textContent` Input property.

<Source
  code={`
<fudis-dt [textContent]="'This is Term element text'" />
<fudis-dd [textContent]="'Here is the Details element text'" />`}
/>

Term element has also `subHeading` Input for adding optional sub-heading between Term and Details elements. Single Description List Item can have multiple Details elements if necessary.

<Source
  code={`
<fudis-dl-item>
  <fudis-dt [subHeading]="'I am Term sub heading'" [textContent]="'I am the main Term text content'" />
  <fudis-dd [textContent]="'I am first Details element'" />
  <fudis-dd [textContent]="'I am second Details element'" />
  <fudis-dd [textContent]="'I am third Details element'" />
</fudis-dl-item>`}
/>

### With Language Badge Group

Description List with Language Badge Group displays Details elements with translations. It's good to note that Description List with Language Badge Group is designed to be used only with Description List **regular** variant.

How to use:

1. Description List Item Term (fudis-dt) element takes a boolean value `languages` and by setting it to `true`, Description List will render Language Badge Group next to the Term element.
2. In order to link Language Badge with selected translation, Description List Item Detail (fudis-dd) has `lang` input, which will specify the language used in Details element content. `lang` input uses `FudisLanguageAbbr` type, which has predefined languages of English, Finnish and Swedish. When language option is used, all of the fudis-dd elements need a suited lang input. If any of the three predefined `lang` inputs are missing, Fudis Language Badge Group will display this translation as missing.

### Accessibility

- Description List component wraps list content into `dl` element
- Each description term is contained within a `dt` element
- Each description details is contained in one or more `dd` element
- `dd` elements are placed directly after `dt` element
- If only one description list item is present, it is converted to paragraph element
