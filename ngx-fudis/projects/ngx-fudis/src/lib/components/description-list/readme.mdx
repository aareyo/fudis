# DescriptionList Component

Fudis DescriptionList component generates an item list from an array of objects type IFudisDescriptionListItem. Each description item has a key value pair. Value can also contain an optional sub-header value.

## Examples

Fudis descriptionList has two different variants 'regular' and 'compact'. Fudis description list's default behavior is to align with fudis grid. This feature can be turned off by setting `disableGrid` as true.
