# Description List Component

Fudis Description List takes a list of objects as a data input. Each list item contains key value pair. This value can also contain an optional sub-header value. Alternatively Fudis Description List can be rendered throught content projection, as shown in following code example:

## Examples

Fudis Description List has two different variants, default 'regular' list and 'compact list option. **Fudis Description extends Fudis Grid Directive and it can take all same inputs as Fudis Grid for an alignment.** Fudis Grid can be disabled in Fudis Description List by setting `disableGrid` as true.

### Accessibility

- Fudis Description List component wraps list content into dl element
- Each list item term is being described is contained within a dt element
- Each description term is contained in one or more dd element
- Description detail elements dd are placed directly after description term dt element
