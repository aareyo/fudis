## Usage Guidelines

Link Component **requires** either `href` or if using Angular Router `routerLinkUrl` with `fragmentId` properties.

Link can be set to open in a new tab with `external` boolean.

Link URL is set as the link title by default. By adding `linkTitle` property the visible title can be modified.

Link color can be adjusted between `primary-dark` (default), `gray-dark` and `white`. Use `gray-dark` and `white` with caution.

### Accessibility

- Link has visible focus
- Icon indicates new tab for visible users but is hidden from screen readers
- Screenreader is informed if link will open in a new tab
