## Usage Guidelines

### Properties

Link component requires either `href` or if using Angular Router `routerLinkUrl` with `fragmentId` properties.

Link can be set to open in a new tab with `isExternalLink` boolean.

Link URL is set as the link title by default, adding `linkTitle` the visible title can be modified.

Link color can be adjusted between _primary-dark_, _default_ and _white_. _Primary-dark_ uses application root color. Use _default_ (gray) and _white_ with caution.

### Accessibility

- Screenreader is informed if link will open in a new tab
- Icon indicates new tab for visible users
