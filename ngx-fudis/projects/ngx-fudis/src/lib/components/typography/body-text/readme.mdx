# Body text

Body text is essentially a paragraph element which inherits Fudis font-family.
There are five size options available: `l-regular`, `l-light`, `m-regular`, `m-light`, `s-regular`.
In addition, optional margin bottom can be added to each individual body text element if needed.

More information about typography can be found under [Foundations/Typography](/docs/foundations-typography--page)
