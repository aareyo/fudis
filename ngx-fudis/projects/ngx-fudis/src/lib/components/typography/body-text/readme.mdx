# Body text

Body text is essentially a paragraph element which inherits Fudis font-family.
There are five size options available: `lg-regular`, `lg-light`, `md-regular`, `md-light` and `sm-regular`.

More information about typography can be found under [Foundations/Typography](/docs/foundations-typography--documentation)

## Usage Guidelines

In addition to paragraph size, fudis-body-text can be modified through `align` and `color` properties. [Fudis spacing directive](/docs/directives-spacing--documentation) is applicable to body-text component as well.

```
<fudis-body-text
	[align]="'center'" 
	[color]="'white'" 
	fudisSpacing [marginBottom]="'xs'" />
```

### Accessibility

- Text color `#484848` passes WCAG AAA requirement against white `#FFFFFF` background
  - Contrast ratio 9.1:1
