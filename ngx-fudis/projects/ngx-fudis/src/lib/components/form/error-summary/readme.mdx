# Error Summary

This component is internally bundled with Fudis Form.

## Attributes

Visibility and attributes of this component are set with Fudis Form's `@Input` directives

### errorSummaryVisible (boolean)

To set Error Summary is visibility. Usually set `true` on click of form submit button. Should be used with [Reload Errors](/docs/services-error-summary--documentation).

### errorSummaryHelpText (string)

Mandatory help text displayed in Error Summary before listing individual errors.

### errorSummaryLiveRemove (boolean)

By default ErrorSummary "refreshes" only when calling [Reload Errors](/docs/services-error-summary--documentation). But when this property is set to `true`, Error Summary removes errors dynamically if user corrects invalid fields.

**Note!** This will NOT dynamically ADD new errors to Error Summary; it only removes those which are listed when reload was called.

### errorSummaryLinkType ('router' | 'href')

Type of the clickable error link in Error Summary. Defaults to Angular router link.
