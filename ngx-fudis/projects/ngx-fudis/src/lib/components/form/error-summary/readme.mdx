# Error Summary

Error Summary is internally bundled with Fudis Form.
It will appear as Fudis Notification with links in the header section of Fudis Form when invalid form is submitted.
All the necessary properties are given through [Fudis From](/docs/components-form-form--documentation) `@Input` directives.

## Properties

### errorSummaryVisible (boolean)

To set Error Summary visibility. By default set to `false` and is meant to set `true` on form submit. Should be used with [Reload Errors](/docs/services-error-summary--documentation).

### errorSummaryHelpText (string)

Mandatory help text displayed in Error Summary before listing individual errors.

### errorSummaryUpdate

Error Summary has three update strategies:

- `reloadOnly` (default): Error Summary list updates only when calling [Reload Errors](/docs/services-error-summary--documentation).
- `onRemove`: Updates Error Summary on reload and removes errors when user corrects one of the visibile errors.
- `all`: Updates Error Summary dynamically when errors are added or removed or reload is called

### errorSummaryLinkType ('router' | 'href')

Type of the clickable error link in Error Summary. Defaults to Angular router link.
