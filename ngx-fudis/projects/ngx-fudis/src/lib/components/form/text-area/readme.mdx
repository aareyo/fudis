# Text Area

Basic Text Area component. 

Text Area can only be resized vertically, horizontal resizing is disabled to avoid layout breaking. Additional features include possibility to add tooltip to the label and maximum character indicator to the bottom right corner.

## Usage Guidelines

### Mandatory Properties

Necessary properties for Text Area are `label` and `control`.

### Validators and Error Handling

Fudis form components use Fudis Validators. See the full documentation on [how to use Fudis Validators](/docs/utilities-validators--documentation). 

If a custom validation is used, read [how to add Error message](/docs/components-form-error-message--documentation).

**Text Area validators**
- required
- minLength and maxLength

#### Text Area `maxLength` Indicator

[MaxLength](/docs/utilities-validators--documentation&args=#minlength-and-maxlength-validators) validator on Text Area does not prompt any error message but disable user to input more than allowed length of characters to the input field.
Additionally, when maxLength validation is used, by passing maxLength Input <code>[maxLength]="maxLength"</code> to Text Area, will activate the maximum character indicator to the bottom right corner.

## Accessibility

- Label and guidance are linked to the Text Area via id
- Screen reader is informed when maximum character length has 5 characters left and when maximum character count is reached
