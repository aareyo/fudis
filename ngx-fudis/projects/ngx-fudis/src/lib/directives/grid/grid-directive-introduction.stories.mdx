# Grid & Grid Item Directives

Grid and Grid Item Directives provide native CSS Grid properties to binded elements and components. Fudis also provide [Grid and Grid Item wrapper components](/docs/components-grid--documentation), which already implement these directives.

In component documentations are more throughout examples, guidelines etc. but same principles apply to any other element directives are added to.
