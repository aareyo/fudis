# Spacing tokens

Following REM values are based on browser's 16px default values.

Figma file for spacing tokens: [Figma file](https://www.figma.com/file/a6G6ZD6W3tlq0qmUkdvdp1/Fudis-DS-Foundations).

| Name                | Value   | Default px value |
|---------------------|---------|------------------|
| $fudis-spacing-zero | 0       | 0                |
| $fudis-spacing-xxs  | 0.25rem | 4px              |
| $fudis-spacing-xs   | 0.5rem  | 8px              |
| $fudis-spacing-s    | 1rem    | 16px             |
| $fudis-spacing-m    | 1.5rem  | 24px             |
| $fudis-spacing-l    | 2rem    | 32px             |
| $fudis-spacing-xl   | 2.5rem  | 40px             |
| $fudis-spacing-xxl  | 4rem    | 64px             |
