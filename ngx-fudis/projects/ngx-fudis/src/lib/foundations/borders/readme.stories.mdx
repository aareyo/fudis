# Border mixins

When using border mixins load them with `@use '../path/to/foundations/borders/mixins.scss' as borders;` to the component's .scss file.

## Border
Set the border of an element.

````
@include borders.border($width, $style, $color)
````

| $width | $style | $color|
| ---- | ---- | ---- |
| string value from spacing tokens map (_$pixel-tokens_) | string value from border tokens map (_$border-styles_) | string value from colors tokens map (_$colors_) |

````
@include borders.border('2px', 'solid', 'primary');	
````


## Outline
Set the outline of an element.

````
@include borders.outline($width, $style, $color)
````

| $width | $style | $color|
| ---- | ---- | ---- |
| string value from spacing tokens map (_$pixel-tokens_) | string value from border tokens map (_$border-styles_) | string value from colors tokens map (_$colors_) |

````
@include borders.outline('1px', 'dashed', 'gray-middle');
````

## Border radius
Set border radius of an element.

````
@include borders.border-radius($width)
````

| $width |
| ---- | 
| string value from spacing tokens map (_$pixel-tokens_) |

````
@include borders.border-radius('2px');
````
