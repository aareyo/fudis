# Border mixins

When using border mixins load them with `@use '../../../foundations/borders/mixins.scss' as borders;` to the component's .scss file.

## Border
Set the border of an element.

````
@include borders.fudis-border($width, $style, $color)
````

| $width | $style | $color|
| ---- | ---- | ---- |
| value from spacing tokens (_$fudis-pixel-tokens_) | value from border tokens (_$fudis-border-styles_) | value from colors tokens (_$fudis-colors_) |

````
@include borders.fudis-border('2px', 'solid', 'primary');	
````


## Outline
